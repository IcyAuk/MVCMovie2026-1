# MVC Movie 2026 #1
A full stack application for managing movies built using the ASP.NET Core MVC framework.

[![Docker Image](https://img.shields.io/badge/Docker-Ready-blue?logo=docker)](https://hub.docker.com/repository/docker/icyauk/mvcmovie/general)

## Key Features
- CRUD Operations for movie records
- Search & Filtering by genre
- Data Validation using Data Annotations
- DB Management using EF Core with SQLite
- Docker-ready

## Tech Stack
- Frontend: Razor Views (HTML/CSS, JS, Bootstrap)
- Database: Entity Framework Core, SQLite
- Backend: ASP.NET Core 9.0.11
- DevOps: Docker https://hub.docker.com/repository/docker/icyauk/mvcmovie/general

## How to Run
- Clone the repo
- Run via Docker: ```docker compose up --build```
- Or via CLI: ```dotnet run```
